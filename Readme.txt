# Diwakar Portfolio

This is a personal portfolio website project.

## Features

- Responsive design
- Portfolio showcase
- Services and skills sections
- Contact form

## Getting Started

1. Clone or download the repository.
2. Open `index.html` in your browser.
3. Customize content in the HTML and `assets/` folders.

## Project Structure

```
index.html
portfolio-details.html
service-details.html
assets/
  css/
  img/
  js/
  scss/
  vendor/
forms/
```
